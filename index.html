Hexo分类与标签配置

发表于 2017-08-09 | 分类于 编程

新建一个页面

首先，我们通过hexo n “name”命令来新建一个页面，在source/_posts目录下找到刚才新建的name.md文件，用记事本或者sublime text打开。
我们看到默认的页面是这样的：

可以编辑标题、日期、标签和内容，但是没有分类的选项。我们可以手动加入 categories:项，或者打开scaffolds/post.md文件，在tages:上面加入categories:,保存后，重新执行hexo n ‘name’命令，会发现新建的页面里有categories:项了。
scaffolds目录下，是新建页面的模板，执行新建命令时，是根据这里的模板页来完成的，所以可以在这里根据你自己的需求添加一些默认值。

添加标签

hexo new page tags
确认站点配置文件_config.yml里有tag_dir: tags
确认主题配置文件_config.yml里有tags: /tags
编辑站点的source/tags/index.md，添加

添加分类

hexo new page categories
确认站点配置文件里有category_dir: categories
确认主题配置文件里有categories: /categories
编辑站点的source/categories/index.md，添加

冒号后面要有空格
分类和标签应该在---之间，---下面是页面内容
要添加tags和categories,主题和站点的_config.yml配置文件tags和categories的注释要去掉。
